# KiCad Project

Primary design source for the OSSM-RS powerboard.

## Main Files

- `pb-v1.kicad_sch`: Schematic
- `pb-v1.kicad_pcb`: PCB layout
- `pb-v1.kicad_pro`: KiCad project file
- `pb-v1.csv`: Consolidated BOM/export snapshot

## Project Libraries

- `project_symbols/`: Custom symbol libraries
- `project_footprints/`: Custom footprints
- `project_3dshapes/`: 3D models used by footprints

# project_3dshapes

3D CAD models referenced by custom footprints in this project.

## Contents

This folder includes STEP/IGS/WRL models for:
- Input connectors
- Inductor package
- Regulator or other mechanical-critical parts

## Notes

- Keep model filenames stable once footprints reference them.
- If replacing a model, verify orientation and origin in PCB 3D viewer.
- Prefer STEP for mechanical exchange and keep WRL only when needed by KiCad view pipeline.
